# IHM
Projet : Figure Geom
Année de création : 2020

Ce projet a été développé par Anne-Sophie Joscht et Laura Masmela Castano.
Contexte : le développement s'est déroulé dans le cadre de la formation DUT Info, dans le module IHM.

Description : ceci est une application pour la création, édition et suppression de figures géométriques avec une architecture MVC.
Pour l'interface graphique, le projet intègre les bibliothèques Awt et Swing de Java.
